Cyphr is a Python based tool for Cryptography.
Cyphr can Encrypt and Decrypt text in 5 different methods:
- ROT13.
- A1Z26 Cipher.
- Caesar Cipher.
- Vigenere Cipher.
- Bacon Cipher.
